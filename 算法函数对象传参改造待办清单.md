## 三、每个函数的修改步骤

对每个待改函数，执行以下步骤：

1. **添加 import**：在算法文件头部添加 `from app.schemas.request_data.xxx_param import XxxParam`
2. **改函数签名**：`def cal_result_xxx(标量参数)` → `def cal_result_xxx(param: XxxParam)`
3. **添加解构代码**：在函数体开头添加 `字段 = param.字段` 的解构语句
4. **修改控制器**：控制器中改为 `cal_result_xxx(param)` 直传对象
5. **验证 lint**：检查无新增错误
