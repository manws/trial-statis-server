## P0 级别修复（公式错误 — 导致计算结果不可信）

### 修复 1：偏度标准误缺少因子 6

- **文件**: `app/stats/base/base_stats_normal.py:180`
- **修复前**: `se_skewness = sqrt(n(n-1) / ((n-2)(n+1)(n+3)))`
- **修复后**: `se_skewness = sqrt(6*n*(n-1) / ((n-2)(n+1)(n+3)))`
- **影响**: 修复前 Z 值偏大约 2.45 倍，所有偏度检验 P 值严重偏小

### 修复 2：alpha_01 临界值错误

- **文件**: `app/stats/base/base_stats_normal.py:290`
- **修复前**: `z > 1.645`（对应 α=0.10）
- **修复后**: `z > 2.576`（对应 α=0.01）
- **影响**: α=0.01 水平的正态性推断结论完全错误

### 修复 3：Yates 连续性校正公式 + 除零保护

- **文件**: `app/stats/chi/chi_stats_4.py:105,102`
- **修复前**: `n * (|ad-bc| - 0.5)² / ((a+b)(c+d)(a+c)(b+d))`
- **修复后**: `n * (|ad-bc| - n/2)² / denom`，并在 denom=0 时抛出 ValueError
- **影响**: Yates 校正后的卡方值和 P 值计算错误；行列合计为 0 时除零崩溃

### 修复 4：Cochran's Q 统计量公式

- **文件**: `app/stats/cq/cq_stats_cq.py:114-132`
- **修复前**: `Q = (n_subjects-1) * Σ(Lj-L̄)² / (ΣCi - ΣCi²/n_subjects)`
- **修复后**: `Q = (k-1) * [k*ΣLj² - T²] / [k*T - ΣCi²]`（标准公式）
- **影响**: Q 统计量的系数和分母均错误，计算结果完全不可信

### 修复 5：Mann-Whitney U 结修正方差分母

- **文件**: `app/stats/rs/rs_stats_indep.py:123` + `app/stats/rs/rs_stats_od1.py:124`
- **修复前**: `Var_U -= tie_correction * n1*n2 / (12*(total_n-1))`
- **修复后**: `Var_U -= tie_correction * n1*n2 / (12*total_n*(total_n-1))`
- **影响**: 有结数据时方差被高估，P 值不准确

### 修复 6：Kaplan-Meier 生存分析改为标准乘积极限

- **文件**: `app/stats/sur/sur_stats_km1.py:130-153` + `app/stats/sur/sur_stats_km2.py:149-167`
- **修复前**: `S(t) = exp(-cumulative_hazard)`（Nelson-Aalen 估计）
- **修复后**: `S(t) = S(t-1) * (1 - d/n)`（标准 KM 乘积极限估计）
- **影响**: 小样本下生存概率估计偏差显著

### 修复 7：聚类分析 linkage 输入格式

- **文件**: `app/stats/clu/clu_stats_q.py:254-255` + `app/stats/clu/clu_stats_r.py:267-268`
- **修复前**: `linkage(distance_matrix_方阵)` — linkage 将方阵误当作原始数据
- **修复后**: `linkage(squareform(distance_matrix))` — 先转换为压缩距离向量
- **影响**: 聚类结果完全错误

## P1 级别修复（边界条件崩溃 — 特定输入导致运行时异常）

### 修复 8：相对风险比 x1=0 时 log(0) 崩溃

- **文件**: `app/stats/base/base_stats_bino4.py:176-188`
- **修复前**: p1=0 时直接执行 `math.log(0)` → `ValueError: math domain error`
- **修复后**: p1=0 时返回 `RR=0.0, log_rr_se=inf, lower=0.0, upper=inf`
- **影响**: 样本 1 阳性数为 0 时崩溃

### 修复 9：泊松概率手动阶乘大数溢出

- **文件**: `app/stats/base/base_stats_pois1.py:96`
- **修复前**: `math.exp(-mu) * (mu**x) / math.factorial(x)` — x>170 时溢出
- **修复后**: `poisson.pmf(x, mu)` — scipy 内部用对数计算，无溢出
- **影响**: 泊松参数较大时崩溃

### 修复 10：McNemar 检验 b+c=0 时误判

- **文件**: `app/stats/chi/chi_stats_paired.py:107-119`
- **修复前**: b+c=0 时 `χ²=inf, p=0.0`（判为显著差异）
- **修复后**: b+c=0 时 `χ²=0.0, p=1.0`（判为完全一致，无差异）
- **影响**: 完全一致的配对数据被误判为显著差异

### 修复 11：F 检验方差为零时除零

- **文件**: `app/stats/hv/hv_stats_f.py:107-119`
- **修复前**: var2=0 时 `var1/var2` → `ZeroDivisionError`
- **修复后**: 两组都为 0 时 F=1.0；仅一组为 0 时抛出 ValueError 明确提示
- **影响**: 某组数据全相同时崩溃

### 修复 12：频数统计四舍五入硬编码 2 位小数

- **文件**: `app/stats/base/base_stats_freq.py:194-195`
- **修复前**: `Decimal('0.01')` 硬编码，忽略 `decimal_places` 参数
- **修复后**: `Decimal('0.' + '0'*decimal_places)` 动态构造
- **影响**: 传入 decimal_places≠2 时四舍五入位数错误
