# 临床Fisher精确检验统计分析模块 (Clinical Fisher Exact Test Statistics Module)

## 模块概述

临床Fisher精确检验统计分析模块提供全面的Fisher精确检验统计分析功能，用于临床数据中2×2列联表的精确概率计算，是临床试验数据分析的重要组成部分。Fisher精确检验在医学研究中广泛应用，特别是在小样本或理论频数较小的情况下，如罕见疾病研究、小规模临床试验等。

## 模块功能

### 1. Fisher精确检验
- 计算2×2列联表的精确P值
- 适用于小样本数据

### 2. 优势比计算
- 计算优势比(OR)及其精确置信区间
- 提供比值比的精确估计

### 3. 边缘合计分析
- 分析行合计和列合计
- 提供表格结构信息

### 4. 概率分布
- 计算所有可能表格的概率
- 识别比观察到的表格更极端的情况

### 5. 结果解释
- 提供统计和临床意义的解释
- 验证结果的准确性

## 临床应用价值

- **小样本研究**：在样本量较小的情况下提供精确检验
- **罕见疾病研究**：分析罕见疾病的发生率差异
- **临床试验**：在早期临床试验中分析疗效差异
- **质量控制**：在小批量质量检测中评估差异

## 统计方法选择指南

### 1. Fisher精确检验适用条件
- 样本量较小（n<40）
- 理论频数<1
- 或存在超过1/5的理论频数<5

### 2. 与卡方检验的比较
- 当满足卡方检验条件时，两者结果相近
- Fisher检验提供精确P值，卡方检验提供近似值

### 3. 临床应用场景
- 小规模临床试验的疗效比较
- 罕见不良反应的发生率比较
- 小样本诊断试验的准确性比较

## 主要函数说明

### 超几何分布函数 - calculate_hypergeometric_pmf

计算超几何分布的概率质量函数值。在临床研究中，这用于计算给定边缘合计下特定表格配置的概率。

**参数**：
- `k`: 成功抽取的数量
- `M`: 总体大小
- `n`: 总体中的成功数
- `N`: 抽样数量

**返回值**：
- `float`: 超几何分布的概率值

### Fisher精确检验函数 - fisher_exact_test

执行Fisher精确检验。在临床研究中，这用于计算2×2列联表的精确P值，以评估两个分类变量之间的关联性。

**参数**：
- `a`: 样本1发生数
- `b`: 样本1未发生数
- `c`: 样本2发生数
- `d`: 样本2未发生数

**返回值**：
- `Dict[str, Any]`: 包含检验统计量和P值的字典

### 生成可能表格函数 - generate_possible_tables

生成给定边缘合计的所有可能表格。在临床研究中，这用于确定在给定边缘合计下所有可能的表格配置。

**参数**：
- `row1_sum`: 第一行合计
- `row2_sum`: 第二行合计
- `col1_sum`: 第一列合计
- `col2_sum`: 第二列合计

**返回值**：
- `List[Dict[str, Any]]`: 所有可能表格的列表

### 概率分析函数 - analyze_tables_probabilities

分析表格概率，找出比观察到的表格更极端的情况。在临床研究中，这用于计算比观察到的表格更极端的所有表格的概率之和。

**参数**：
- `tables`: 所有可能表格的列表
- `observed_prob`: 观察到的表格的概率

**返回值**：
- `Dict[str, Any]`: 概率分析结果

### 完整检验流程函数 - perform_fisher_exact_test

执行完整的Fisher精确检验流程。在临床研究中，这用于提供全面的Fisher精确检验结果，包括边缘合计、可能表格分析和统计解释。

**参数**：
- `a`: 样本1发生数
- `b`: 样本1未发生数
- `c`: 样本2发生数
- `d`: 样本2未发生数

**返回值**：
- `Dict[str, Any]`: 包含完整检验结果的字典

### 结果计算函数 - cal_result_fisher

生成Fisher精确检验统计分析的完整报告字典。此函数整合了Fisher精确检验的所有关键指标，生成标准化的字典格式报告，适用于临床研究报告的需求，提供全面的Fisher精确检验结果。报告包括输入参数、观察表格、所有可能表格、检验结果、验证结果和统计解释等信息，便于临床医生和研究人员快速理解Fisher精确检验的特征。

**参数**：
- `param`: ChiSquareParamFisher对象，包含a, b, c, d

**返回值**：
- 包含Fisher精确检验统计分析指标的字典，包括：
  - `table_name`: 报告表格名称
  - `input_parameters`: 输入参数信息
  - `observed_table`: 观察表格信息
  - `all_tables`: 所有可能表格
  - `test_results`: 检验结果
  - `scipy_verification`: scipy验证结果
  - `interpretation`: 统计解释
  - `remark`: 备注信息

## 结果解读注意事项

1. **P值解释**：精确概率，表示在给定边缘合计下观察到当前或更极端结果的概率
2. **优势比解释**：衡量暴露与结局之间关联强度的指标
3. **显著性水平**：通常使用α=0.05作为判断标准
4. **临床意义**：统计显著性不等同于临床重要性
5. **适用条件**：特别适用于小样本或理论频数较小的情况

## 使用示例

```python
from app.stats.chi.chi_stats_fisher import cal_result_fisher
from app.schemas.request_data.chi_param import ChiSquareParamFisher

# 创建参数对象
param = ChiSquareParamFisher(
    a=5,    # 样本1发生数
    b=15,   # 样本1未发生数
    c=2,    # 样本2发生数
    d=18    # 样本2未发生数
)

# 计算Fisher精确检验
result = cal_result_fisher(param)

# 查看结果
print(f"输入参数: {result['input_parameters']}")
print(f"观察表格: {result['observed_table']}")
print(f"检验结果: {result['test_results']}")
print(f"统计解释: {result['interpretation']}")
```
