# 临床卡方P值计算统计分析模块 (Clinical Chi-square P-value Calculation Statistics Module)

## 模块概述

临床卡方P值计算统计分析模块提供全面的卡方P值计算统计分析功能，用于临床数据中卡方统计量的P值计算，是临床试验数据分析的重要组成部分。卡方P值计算在医学研究中广泛应用，如评估分类数据间的关联性、检验独立性等。

## 模块功能

### 1. P值计算
- 根据卡方统计量和自由度计算P值
- 提供精确的概率计算

### 2. 临界值计算
- 计算给定显著性水平的临界值
- 提供决策依据

### 3. 统计解释
- 提供统计显著性解释
- 评估检验结果

### 4. 结果汇总
- 生成完整的分析报告
- 提供多种效应量指标

## 临床应用价值

- **关联性分析**：评估分类变量间的关联性
- **独立性检验**：检验两个分类变量是否独立
- **拟合优度**：检验观察频数与理论频数的拟合程度
- **结果验证**：验证已有卡方统计量的显著性

## 统计方法选择指南

### 1. 卡方检验适用条件
- 样本量足够大
- 理论频数不太小（一般要求≥5）
- 分类变量间相互独立

### 2. 临床应用场景
- 比较两个分类变量的分布
- 检验暴露因素与疾病的关系
- 分析治疗效果与分类指标的关联

## 主要函数说明

### 卡方P值计算函数 - chi_square_p_value

根据卡方值和自由度计算P值。在临床研究中，这用于根据已知的卡方统计量和自由度计算相应的P值，以判断统计结果的显著性。

**参数**：
- `chi_square`: 卡方统计量值
- `df`: 自由度

**返回值**：
- `Dict[str, Any]`: 包含P值计算结果的字典

### 完整检验流程函数 - perform_chi_square_p_test

执行卡方P值计算的完整分析。在临床研究中，这用于执行完整的卡方P值计算过程，以提供全面的统计分析结果，便于临床医生和研究人员进行全面评估。

**参数**：
- `chi_square`: 卡方统计量值
- `df`: 自由度

**返回值**：
- `Dict[str, Any]`: 包含完整分析结果的字典

### 结果计算函数 - cal_result_chi_p

生成卡方P值计算统计分析的完整报告字典。此函数整合了卡方P值计算的所有关键指标，生成标准化的字典格式报告，适用于临床研究报告的需求，提供全面的卡方P值计算结果。报告包括输入参数、检验结果、显著性检验和统计解释等信息，便于临床医生和研究人员快速理解卡方P值计算的特征。

**参数**：
- `param`: ChiSquareParamP对象，包含chi, df

**返回值**：
- 包含卡方P值计算统计分析指标的字典，包括：
  - `table_name`: 报告表格名称
  - `input_parameters`: 输入参数信息
  - `test_results`: 检验结果
  - `significance_tests`: 显著性检验结果
  - `interpretation`: 统计解释
  - `remark`: 备注信息

## 结果解读注意事项

1. **P值解释**：P值越小表示证据越强，拒绝原假设的可能性越大
2. **显著性水平**：通常使用α=0.05作为判断标准
3. **临床意义**：统计显著性不等同于临床重要性
4. **检验前提**：注意卡方检验的应用条件
5. **多重比较**：当进行多次检验时需要注意调整显著性水平

## 使用示例

```python
from app.stats.chi.chi_stats_p import cal_result_chi_p
from app.schemas.request_data.chi_param import ChiSquareParamP

# 创建参数对象
param = ChiSquareParamP(
    chi=4.85,  # 卡方统计量值
    df=1       # 自由度
)

# 计算卡方P值
result = cal_result_chi_p(param)

# 查看结果
print(f"输入参数: {result['input_parameters']}")
print(f"检验结果: {result['test_results']}")
print(f"显著性检验: {result['significance_tests']}")
print(f"统计解释: {result['interpretation']}")
```
