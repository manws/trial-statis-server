# 临床配对资料卡方检验统计分析模块 (Clinical Paired Data Chi-Square Test Statistics Module)

## 模块概述

临床配对资料卡方检验统计分析模块提供全面的配对资料卡方检验统计分析功能，用于临床数据中配对分类变量的比较，是临床试验数据分析的重要组成部分。配对卡方检验在医学研究中广泛应用，如比较同一组患者在治疗前后的分类指标变化、比较两种诊断方法的一致性等。

## 模块功能

### 1. McNemar检验
- 计算配对卡方统计量
- 评估配对数据的差异性

### 2. 连续性校正
- 对于小样本提供Yates校正
- 减少第一类错误的发生

### 3. Kappa一致性系数
- 评估两种方法的一致性
- 考虑机会一致性的调整

### 4. 优势比计算
- 计算配对数据的优势比
- 提供效应量估计

### 5. 不一致分析
- 分析不一致对的分布
- 评估不一致的程度

## 临床应用价值

- **诊断比较**：比较两种诊断方法的一致性
- **前后对照**：比较同一组患者治疗前后的分类指标
- **方法验证**：评估新方法与金标准的一致性
- **质量控制**：评估重复测量的一致性

## 统计方法选择指南

### 1. McNemar检验适用条件
- 配对二分类数据
- 数据呈正态分布或大样本
- 每个个体测量两次

### 2. Kappa系数解释
- Kappa > 0.75：一致性良好
- 0.4 ≤ Kappa ≤ 0.74：一致性一般
- Kappa < 0.4：一致性较差

### 3. 临床应用场景
- 比较两种诊断试验的准确性
- 评估治疗前后分类指标变化
- 比较不同评估方法的一致性

## 主要函数说明

### McNemar检验函数 - mcnemar_test

执行McNemar检验。在临床研究中，这用于比较配对二分类数据的差异性，以评估两种方法或时间点之间的统计学显著性。

**参数**：
- `b`: 样本1阳性/成功，样本2阴性/失败的配对数
- `c`: 样本1阴性/失败，样本2阳性/成功的配对数

**返回值**：
- `Dict[str, Any]`: 包含检验统计量和P值的字典

### Kappa系数计算函数 - kappa_coefficient

计算Kappa一致性系数。在临床研究中，这用于评估两种分类方法的一致性，考虑了偶然一致性的因素。

**参数**：
- `a`: 样本1阳性/成功，样本2阳性/成功的配对数
- `b`: 样本1阳性/成功，样本2阴性/失败的配对数
- `c`: 样本1阴性/失败，样本2阳性/成功的配对数
- `d`: 样本1阴性/失败，样本2阴性/失败的配对数

**返回值**：
- `Dict[str, Any]`: Kappa系数及相关统计量

### 优势比计算函数 - odds_ratio_calculation

计算优势比(OR)和相对危险度(RR)。在临床研究中，这用于评估配对数据中的关联强度。

**参数**：
- `a`: 样本1阳/成功，样本2阳/成功的配对数
- `b`: 样本1阳/成功，样本2阴/失败的配对数
- `c`: 样本1阴/失败，样本2阳/成功的配对数
- `d`: 样本1阴/失败，样本2阴/失败的配对数

**返回值**：
- `Dict[str, Any]`: OR和RR及相关统计量

### 配对卡方检验函数 - paired_chi_square_test

执行配对资料的卡方检验。在临床研究中，这用于分析配对二分类数据的差异性。

**参数**：
- `a`: 样本1阳/成功，样本2阳/成功的配对数
- `b`: 样本1阳/成功，样本2阴/失败的配对数
- `c`: 样本1阴/失败，样本2阳/成功的配对数
- `d`: 样本1阴/失败，样本2阴/失败的配对数

**返回值**：
- `Dict[str, Any]`: 检验结果字典

### 完整检验流程函数 - perform_paired_chi_square_test

执行完整的配对资料卡方检验流程。在临床研究中，这用于提供全面的配对卡方检验结果，包括不一致分析、McNemar检验、Kappa系数和效应量计算。

**参数**：
- `a`: 样本1阳/成功，样本2阳/成功的配对数
- `b`: 样本1阳/成功，样本2阴/失败的配对数
- `c`: 样本1阴/失败，样本2阳/成功的配对数
- `d`: 样本1阴/失败，样本2阴/失败的配对数

**返回值**：
- `Dict[str, Any]`: 包含完整检验结果的字典

### 结果计算函数 - cal_result_chi_paired

生成配对资料卡方检验统计分析的完整报告字典。此函数整合了配对资料卡方检验的所有关键指标，生成标准化的字典格式报告，适用于临床研究报告的需求，提供全面的配对卡方检验结果。报告包括输入参数、不一致分析、McNemar检验、校正检验、Kappa一致性系数、优势比分析和统计解释等信息，便于临床医生和研究人员快速理解配对卡方检验的特征。

**参数**：
- `param`: ChiSquareParamPaired对象，包含a, b, c, d

**返回值**：
- 包含配对资料卡方检验统计分析指标的字典，包括：
  - `table_name`: 报告表格名称
  - `input_parameters`: 输入参数信息
  - `discordant_analysis`: 不一致分析
  - `mcnemar_test`: McNemar检验结果
  - `corrected_mcnemar_test`: 校正McNemar检验结果
  - `kappa_coefficient`: Kappa一致性系数
  - `odds_ratio`: 优势比分析
  - `interpretation`: 统计解释
  - `remark`: 备注信息

## 结果解读注意事项

1. **P值解释**：在零假设成立的前提下，观察到当前或更极端结果的概率
2. **Kappa系数**：评估一致性强度的指标
3. **显著性水平**：通常使用α=0.05作为判断标准
4. **临床意义**：统计显著性不等同于临床重要性
5. **效应量**：关注Kappa系数和优势比的实际意义

## 使用示例

```python
from app.stats.chi.chi_stats_paired import cal_result_chi_paired
from app.schemas.request_data.chi_param import ChiSquareParamPaired

# 创建参数对象
param = ChiSquareParamPaired(
    a=30,   # 样本1阳/成功，样本2阳/成功
    b=10,   # 样本1阳/成功，样本2阴/失败
    c=15,   # 样本1阴/失败，样本2阳/成功
    d=45    # 样本1阴/失败，样本2阴/失败
)

# 计算配对资料卡方检验
result = cal_result_chi_paired(param)

# 查看结果
print(f"输入参数: {result['input_parameters']}")
print(f"不一致分析: {result['discordant_analysis']}")
print(f"McNemar检验: {result['mcnemar_test']}")
print(f"Kappa系数: {result['kappa_coefficient']}")
print(f"统计解释: {result['interpretation']}")
```
