# 多元线性回归分析 (Multiple Linear Regression Analysis)

## 概述
多元线性回归是一种统计方法，用于分析一个连续因变量与多个自变量之间的线性关系。它扩展了一元线性回归的概念，能够同时考虑多个因素对因变量的影响，是医学研究和数据分析中最重要的统计方法之一。

## 数学原理
多元线性回归模型表示为：
```
Y = β₀ + β₁X₁ + β₂X₂ + ... + βₖXₖ + ε
```
其中：
- Y 是因变量（响应变量）
- X₁, X₂, ..., Xₖ 是自变量（解释变量）
- β₀ 是截距项
- β₁, β₂, ..., βₖ 是各自变量的回归系数
- ε 是随机误差项

回归系数通过最小二乘法估计：β̂ = (X'X)⁻¹X'Y

## 功能特点
1. **回归系数估计**：计算截距和各变量的回归系数及其显著性检验
2. **模型拟合评估**：计算决定系数(R²)、调整R²等指标
3. **统计检验**：执行t检验（系数）和F检验（模型整体）
4. **残差分析**：提供预测值和残差用于模型诊断
5. **结果解释**：提供统计结果的临床意义解释

## 适用条件
- 因变量为连续变量
- 自变量与因变量之间存在线性关系
- 残差服从正态分布
- 残差方差齐性
- 观测值相互独立
- 自变量间不存在多重共线性

## 临床应用场景
- 分析多个因素对疗效的综合影响
- 构建疾病预后预测模型
- 研究多个生物标志物与疾病的关联
- 控制混杂因素的影响

## 输出指标说明
- **回归系数**：截距和各变量系数及其标准误、t值和P值
- **模型统计量**：R²、调整R²、RMSE、F统计量等
- **预测值**：模型预测的因变量值和残差
- **解释**：回归方程、系数解释和模型显著性判断

## 注意事项
- 多元线性回归假设数据满足线性关系、独立性、正态性、方差齐性和无多重共线性
- 需要检查自变量间的多重共线性（VIF、条件指数等）
- 统计显著性不等于临床重要性，需结合专业知识判断
- 避免过度拟合，注意变量选择策略

## 示例
给定年龄(X₁)、BMI(X₂)和收缩压(X₃)数据，分析它们对舒张压(Y)的影响，得到回归方程：Ŷ = 20.5 + 0.15X₁ + 0.8X₂ + 0.3X₃，表明在控制其他变量的情况下，年龄每增加1岁，舒张压平均增加0.15mmHg。
