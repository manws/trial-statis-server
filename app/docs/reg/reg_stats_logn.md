# 多项逻辑回归分析 (Multinomial Logistic Regression Analysis)

## 概述
多项逻辑回归是一种统计方法，用于分析一个多项分类因变量与一个或多个自变量之间的关系。它扩展了二元逻辑回归的概念，适用于因变量有三个或更多类别且这些类别之间没有自然排序关系的情况，广泛应用于医学研究和社会科学中。

## 数学原理
多项逻辑回归模型将一个参考类别（基准类别）与其他类别进行比较。对于有J个类别的因变量，模型表示为：
```
ln[P(Y=j)/P(Y=J)] = β₀ⱼ + β₁ⱼX₁ + β₂ⱼX₂ + ... + βₖⱼXₖ  (j=1,2,...,J-1)
```
其中：
- P(Y=j) 是属于类别j的概率
- 第J类是参考类别（基准类别）
- X₁, X₂, ..., Xₖ 是自变量
- β₀ⱼ, β₁ⱼ, ..., βₖⱼ 是类别j相对于基准类别的回归系数

各类别的概率为：
```
P(Y=j|X) = exp(β₀ⱼ + β₁ⱼX₁ + ... + βₖⱼXₖ) / [1 + Σᵢ₌₁ᴶ⁻¹exp(β₀ᵢ + β₁ᵢX₁ + ... + βₖᵢXₖ)]
P(Y=J|X) = 1 / [1 + Σᵢ₌₁ᴶ⁻¹exp(β₀ᵢ + β₁ᵢX₁ + ... + βₖᵢXₖ)]
```

参数通过最大似然法估计。

## 功能特点
1. **参数估计**：使用最大似然法估计各对比组的回归系数
2. **模型拟合评估**：计算对数似然值、AIC、BIC等指标
3. **统计检验**：执行Wald检验判断各自变量系数是否显著
4. **分类性能评估**：计算准确率和混淆矩阵
5. **结果解释**：提供相对基准类别的优势比解释

## 适用条件
- 因变量为多项分类变量（≥3类），且类别间无自然排序关系
- 自变量可以是连续变量或分类变量
- 观测值相互独立
- 满足独立性无关选项假设（IIA）
- 无完全分离现象

## 临床应用场景
- 预测疾病亚型发生概率
- 分析治疗选择的影响因素
- 识别疾病严重程度的危险因素
- 构建多分类诊断模型

## 输出指标说明
- **回归系数**：各类别相对于基准类别的系数及其标准误、Wald统计量和P值
- **优势比**：相对于基准类别的优势比解释
- **模型统计量**：对数似然值、McFadden's R²、AIC、BIC等
- **分类性能**：准确率和混淆矩阵
- **预测值**：各类别的预测概率和最终分类结果

## 注意事项
- 多项逻辑回归假设数据满足独立性、广义线性关系和独立性无关选项假设（IIA）
- 优势比解释为相对于基准类别的相对优势
- 需要检查IIA假设是否成立（可通过hausman-mcelroy检验）
- 样本量应足够支撑多类别和多参数的估计
- 需要考虑类别不平衡问题

## 示例
研究年龄(X₁)、性别(X₂)和BMI(X₃)对疾病类型(Y)的影响，假设疾病有三种类型（A、B、C），以C型为基准类别。得到A型相对于C型的回归方程：logit(P(A)/P(C)) = -2.1 + 0.03X₁ + 0.7X₂ + 0.05X₃，B型相对于C型的回归方程：logit(P(B)/P(C)) = -1.5 + 0.02X₁ + 0.5X₂ + 0.04X₃。这表明在控制其他变量的情况下，年龄每增加1岁，患A型疾病相对于C型的优势比增加exp(0.03)=1.03倍。
