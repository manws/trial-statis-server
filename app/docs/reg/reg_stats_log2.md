# 二元逻辑回归分析 (Binary Logistic Regression Analysis)

## 概述
二元逻辑回归是一种统计方法，用于分析一个二分类因变量与一个或多个自变量之间的关系。它通过估计事件发生的概率来建模，广泛应用于医学研究和流行病学中，如预测疾病发生概率、分析危险因素等。

## 数学原理
二元逻辑回归模型表示为：
```
logit(p) = ln[p/(1-p)] = β₀ + β₁X₁ + β₂X₂ + ... + βₖXₖ
```
其中：
- p 是事件发生的概率
- logit(p) 是对数几率函数
- X₁, X₂, ..., Xₖ 是自变量
- β₀, β₁, ..., βₖ 是回归系数

事件发生的概率为：
```
p = exp(β₀ + β₁X₁ + ... + βₖXₖ) / [1 + exp(β₀ + β₁X₁ + ... + βₖXₖ)]
```

参数通过最大似然法估计，使用迭代方法求解。

## 功能特点
1. **参数估计**：使用最大似然法估计回归系数
2. **模型拟合评估**：计算对数似然值、AIC、BIC等指标
3. **统计检验**：执行Wald检验判断各自变量系数是否显著
4. **分类性能评估**：计算准确率、敏感性、特异性等指标
5. **结果解释**：提供优势比解释和统计显著性判断

## 适用条件
- 因变量为二分类变量（0/1）
- 自变量可以是连续变量或分类变量
- 观测值相互独立
- 不存在完全分离现象
- 线性关系假设（在对数几率尺度上）

## 临床应用场景
- 预测疾病发生风险
- 分析治疗成功的概率
- 识别疾病危险因素
- 构建诊断预测模型

## 输出指标说明
- **回归系数**：截距和各变量系数及其标准误、Wald统计量和P值
- **优势比**：exp(β)表示自变量每增加一个单位，事件发生与不发生比值的变化倍数
- **模型统计量**：对数似然值、McFadden's R²、AIC、BIC等
- **分类性能**：准确率、敏感性、特异性、精确率及混淆矩阵
- **预测值**：预测概率和分类结果

## 注意事项
- 逻辑回归假设数据满足独立性、线性关系（在对数几率尺度）和无完全分离
- 优势比解释需要特别注意，OR > 1表示正相关，OR < 1表示负相关
- 需要注意样本量是否足够支撑所含变量的数量
- 应评估模型的区分度和校准度

## 示例
研究年龄(X₁)、性别(X₂)和吸烟状况(X₃)对患心脏病(Y)的影响，得到回归方程：logit(p) = -4.5 + 0.05X₁ + 0.8X₂ + 1.2X₃。年龄每增加1岁，患心脏病的优势比增加exp(0.05)=1.05倍；男性相比女性的优势比为exp(0.8)=2.23倍；吸烟者相比非吸烟者的OR为exp(1.2)=3.32倍。
