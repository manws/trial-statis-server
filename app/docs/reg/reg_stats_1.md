# 一元线性回归分析 (Simple Linear Regression Analysis)

## 概述
一元线性回归是一种统计方法，用于分析两个连续变量之间的线性关系。它通过拟合一条直线来描述自变量X对因变量Y的影响，是最基础也是最常用的回归分析方法之一。

## 数学原理
一元线性回归模型表示为：
```
Y = β₀ + β₁X + ε
```
其中：
- Y 是因变量（响应变量）
- X 是自变量（解释变量）
- β₀ 是截距项
- β₁ 是X的回归系数（斜率）
- ε 是随机误差项

回归系数通过最小二乘法估计：
- 斜率: β₁ = Σ[(xi-x̄)(yi-ȳ)] / Σ[(xi-x̄)²]
- 截距: β₀ = ȳ - β₁x̄

## 功能特点
1. **回归系数估计**：计算截距和斜率参数及其显著性检验
2. **模型拟合评估**：计算决定系数(R²)和调整R²
3. **统计检验**：执行t检验和F检验
4. **残差分析**：提供预测值和残差用于模型诊断
5. **结果解释**：提供统计结果的临床意义解释

## 适用条件
- 两个变量均为连续变量
- 变量间存在线性关系
- 残差服从正态分布
- 残差方差齐性
- 观测值相互独立

## 临床应用场景
- 分析药物剂量与疗效的关系
- 研究生理指标间的关联
- 预测患者预后指标
- 探索生物标志物与疾病的关系

## 输出指标说明
- **回归系数**：截距和斜率及其标准误、t值和P值
- **模型统计量**：R²、调整R²、RMSE、F统计量等
- **预测值**：模型预测的因变量值和残差
- **解释**：回归方程、系数解释和模型显著性判断

## 注意事项
- 线性回归假设数据满足线性关系、独立性、正态性和方差齐性
- 统计显著性不等于临床重要性，需结合专业知识判断
- 避免超出数据范围的外推预测

## 示例
给定身高(X)和体重(Y)数据，分析身高对体重的影响，得到回归方程：Ŷ = -80.2 + 0.75X，表明身高每增加1cm，体重平均增加0.75kg。
