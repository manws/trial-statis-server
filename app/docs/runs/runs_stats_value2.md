# 按中位数分类的数值变量游程检验 (Runs Test for Numerical Variables Classified by Median)

## 概述
按中位数分类的数值变量游程检验是一种非参数统计方法，用于检验数值序列的随机性。该方法首先将数值数据按中位数上下分为二分类数据（大于等于中位数为1，小于中位数为0），然后对转换后的二分类数据执行游程检验。

## 数学原理
游程检验基于以下统计量计算：
- 游程数：序列中连续相同值的段落数
- 期望游程数：E(R) = 1 + (2n₀n₁)/(n₀+n₁)，其中n₀和n₁分别为0和1的个数
- 游程方差：Var(R) = (2n₀n₁(2n₀n₁-n))/(n²(n-1))，其中n为总样本量
- 近似正态统计量：Z = (R - E(R))/√Var(R)，在大样本下近似服从标准正态分布

## 功能特点
1. **数据转换**：自动将数值数据按中位数分为两类
2. **游程计算**：计算二分类数据序列中的游程数
3. **统计检验**：执行游程检验并计算P值
4. **随机性评估**：评估数据序列的随机性
5. **结果解释**：提供统计结果的专业解释

## 适用条件
- 数据序列独立
- 二分类变量（通过中位数分类转换）
- 样本量不宜过小（一般要求各类别个数不少于10）
- 适用于检验序列的随机性

## 临床应用场景
- 检验实验数据的随机性
- 验证随机化分组的效果
- 监控实验室数据的质量
- 评估测量过程的稳定性

## 输出指标说明
- **输入参数**：样本量、原始数据中位数、均值、标准差等
- **数据转换信息**：分类方法、阈值、各类别计数等
- **游程统计**：观察游程数、期望游程数、方差等
- **检验统计量**：Z统计量、双侧P值等
- **显著性检验**：不同显著性水平下的结果
- **统计解释**：对结果的专业解释和随机性评估

## 注意事项
- 游程检验假设数据序列独立
- 中位数作为分割点对异常值不敏感，适合偏态分布数据
- 当游程数过多或过少时，可能表示数据序列存在非随机模式
- 统计显著性不代表实际重要性，需结合专业知识判断

## 示例
给定一组血糖测量数据，按中位数5.5mmol/L分为两类（≥5.5为1，<5.5为0），对转换后的二分类序列进行游程检验，评估血糖值的时间序列是否随机。
