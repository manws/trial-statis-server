# 二分类变量游程检验 (Runs Test for Binary Classification Variables)

## 概述
二分类变量游程检验是一种非参数统计方法，用于检验二分类序列的随机性。该方法直接对二分类数据（通常为0和1）进行分析，计算序列中的游程数并执行统计检验，判断数据序列是否表现出随机性。

## 数学原理
游程检验基于以下统计量计算：
- 游程数：序列中连续相同值的段落数
- 期望游程数：E(R) = 1 + (2n₀n₁)/(n₀+n₁)，其中n₀和n₁分别为0和1的个数
- 游程方差：Var(R) = (2n₀n₁(2n₀n₁-n))/(n²(n-1))，其中n为总样本量
- 近似正态统计量：Z = (R - E(R))/√Var(R)，在大样本下近似服从标准正态分布

## 功能特点
1. **游程计算**：计算二分类数据序列中的游程数
2. **统计检验**：执行游程检验并计算P值
3. **随机性评估**：评估数据序列的随机性
4. **结果解释**：提供统计结果的专业解释

## 适用条件
- 数据序列独立
- 二分类变量（仅包含0和1或两个类别）
- 样本量不宜过小（一般要求各类别个数不少于10）
- 适用于检验序列的随机性

## 临床应用场景
- 检验实验数据的随机性
- 验证随机化分组的效果
- 监控实验室数据的质量
- 评估治疗反应序列的模式
- 分析二分类结局（如治愈/未治愈）的随机性

## 输出指标说明
- **table_name**：表格名称，固定为"二分类变量游程检验分析"
- **input_parameters**：输入参数信息
  - sample_size: 样本量，整数类型
  - count_0: 值为0的数据个数，整数类型
  - count_1: 值为1的数据个数，整数类型
- **run_statistics**：游程统计信息
  - observed_runs: 观察到的游程数，整数类型
  - expected_runs: 期望游程数，浮点数类型
  - var_runs: 游程方差，浮点数类型
  - std_runs: 游程标准差，浮点数类型
- **test_statistics**：检验统计量
  - z_statistic: Z统计量，浮点数类型
  - p_value_two_sided: 双侧P值，浮点数类型
  - p_value_range: P值范围描述，字符串类型
- **significance_tests**：显著性检验结果
  - significant_at_05: 0.05水平是否显著，布尔类型
  - significant_at_01: 0.01水平是否显著，布尔类型
- **interpretation**：统计解释
  - randomness_assessment: 随机性评估，字符串类型
- **remark**：备注信息（样本量和各类别数量），字符串类型

## 注意事项
- 游程检验假设数据序列独立
- 当游程数过多或过少时，可能表示数据序列存在非随机模式
- 游程数较少表示数据倾向于聚集（如000111）
- 游程数较多表示数据倾向于交替（如010101）
- 统计显著性不代表实际重要性，需结合专业知识判断

## 示例
给定一组二分类数据序列[0,0,1,1,1,0,1,0,0,1]，检验该序列是否随机产生。计算得到游程数为6，通过统计检验判断该游程数是否与随机序列的期望游程数有显著差异。
