# 临床方差齐性F检验统计分析模块 (Clinical Variance Homogeneity F-Test Statistics Module)

## 模块概述

临床方差齐性F检验统计分析模块提供全面的方差齐性F检验统计分析功能，用于临床数据中方差齐性的检验，是比较两组数据方差是否相等的经典统计方法。F检验在医学研究中广泛应用，如比较两组患者的方差是否存在显著差异、验证方差齐性假设等。

## 模块功能

### 1. F统计量计算
- 计算两组数据方差比的F统计量
- 评估两组方差的差异程度

### 2. 自由度计算
- 计算分子和分母的自由度
- 为F检验提供基础

### 3. 显著性检验
- 执行双侧F检验判断方差是否齐性
- 提供不同显著性水平的判断

### 4. 方差比计算
- 计算两组方差的比值
- 量化方差差异程度

### 5. 统计解释
- 提供结果的临床意义解释
- 帮助理解统计结果的实际含义

## 临床应用价值

- **方差齐性检验**：验证不同组别间方差是否齐性
- **统计方法选择**：为后续统计分析方法的选择提供依据
- **数据质量评估**：评估不同组别间变异程度的差异
- **假设检验验证**：验证统计方法应用的前提条件

## 统计方法选择指南

### 1. F检验适用条件
- 两组数据均服从正态分布
- 数据相互独立
- 适用于小样本情况下的方差比较

### 2. 临床应用场景
- 比较两组患者某项指标的变异性差异
- 验证方差分析前的方差齐性假设
- 检验不同处理组间变异程度的差异

## 主要函数说明

### 方差齐性F检验函数 - calculate_f_test_variance_homogeneity

执行方差齐性F检验（方差比检验）。在临床研究中，这用于检验两组数据的方差是否齐性，即比较两组数据的变异程度是否存在显著差异。

**参数**：
- `data_list`: List[List[float]]，每个子列表代表一组数据

**返回值**：
- 包含F检验统计量和结果的字典

### 结果计算函数 - cal_result_hv_f

生成方差齐性F检验统计分析的完整报告字典。此函数整合了方差齐性F检验的所有关键指标，生成标准化的字典格式报告，适用于临床研究报告的需求，提供全面的F检验结果。

**参数**：
- `data_list`: List[List[float]]，每个子列表代表一组数据

**返回值**：
- 包含方差齐性F检验统计分析指标的字典，包括：
  - `input_parameters`: 输入参数信息
  - `test_statistics`: 检验统计量
  - `significance_tests`: 显著性检验结果
  - `interpretation`: 统计解释

## 结果解读注意事项

1. **F统计量解释**：衡量两组方差的比值，F值越接近1表示方差越齐
2. **P值解释**：在零假设（两组方差相等）成立的情况下，观察到当前或更极端结果的概率
3. **显著性水平**：通常使用α=0.05作为判断方差齐性的标准
4. **临床意义**：方差齐性是许多统计方法的前提条件，不满足时需选用其他方法

## 使用示例

```python
from app.stats.hv.hv_stats_f import cal_result_hv_f

# 创建示例数据，两组数据
data_list = [
    [10.2, 12.1, 9.8, 11.5, 13.2],  # 第一组数据
    [8.7, 9.2, 8.9, 9.5, 10.1]      # 第二组数据
]

# 计算方差齐性F检验
result = cal_result_hv_f(data_list)

# 查看结果
print(f"输入参数: {result['input_parameters']}")
print(f"检验统计量: {result['test_statistics']}")
print(f"显著性检验: {result['significance_tests']}")
print(f"统计解释: {result['interpretation']}")
```
