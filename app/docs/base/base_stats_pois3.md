# 临床泊松分布样本均数与总体均数比较统计分析模块 (Clinical Sample Mean vs Population Mean Comparison for Poisson Distribution Statistics Module)

## 模块概述

临床泊松分布样本均数与总体均数比较统计分析模块提供全面的泊松分布样本均数与总体均数比较统计分析功能，用于临床数据中样本均数与已知总体均数的差异检验，是临床试验数据分析的重要组成部分。样本均数与总体均数比较在医学研究中广泛应用，如评估某段时间内疾病发病率是否显著高于历史平均水平、某种不良事件发生率是否超过安全阈值等。

## 模块功能

### 1. 精确检验
- 使用泊松分布精确计算P值
- 适用于任何样本量

### 2. 正态近似检验
- 使用正态近似法计算统计量和P值
- 适用于大样本情况

### 3. 单侧双侧检验
- 支持左侧、右侧和双侧检验
- 根据研究目的选择合适的方法

### 4. 效应量计算
- 计算差异的效应量
- 评估差异的实际意义

### 5. 检验效能
- 计算检验的效能
- 评估检测能力

## 临床应用价值

- **疾病监测**：监测疾病发病率是否显著升高
- **安全评估**：评估不良事件发生率是否超过标准
- **质量控制**：评估医疗指标是否达到既定标准
- **政策评估**：评估公共卫生措施是否达到预期效果

## 统计方法选择指南

### 1. 精确检验适用条件
- 任何样本量下都适用
- 特别适用于小样本或均数较小的情况

### 2. 正态近似法适用条件
- 样本量较大
- 总体均数足够大（通常λ ≥ 5）

### 3. 临床应用场景
- 传染病发病率是否超出警戒线
- 医院感染率是否显著上升
- 医疗事故率是否超过安全标准

## 主要函数说明

### 泊松分布精确检验函数 - exact_test_poisson

泊松分布精确检验。在临床研究中，这用于检验观察到的事件数与理论事件数之间的差异，通过计算泊松分布的确切概率来判断差异的统计学意义。

**参数**：
- `x`: 观察到的事件数
- `n`: 观察单位数（例如时间长度、人口数量等）
- `lambda0`: 总体均数（零假设下的单位均数）
- `alternative`: 备择假设类型 ('two-sided', 'less', 'greater')

**返回值**：
- `Dict[str, float]`: 包含检验统计量和P值的字典

### 正态近似法检验函数 - normal_approximation_test

正态近似法检验。在临床研究中，这用于检验样本均数与总体均数之间的差异，通过正态近似计算统计量来判断差异的统计学意义。

**参数**：
- `x`: 观察到的事件数
- `n`: 观察单位数
- `lambda0`: 总体均数（零假设下的单位均数）
- `alternative`: 备择假设类型 ('two-sided', 'less', 'greater')

**返回值**：
- `Dict[str, float]`: 包含检验统计量和P值的字典

### 检验效函数 - calculate_power

计算检验效能。在临床研究中，这用于评估当前样本量下检测特定差异的能力，有助于研究设计和结果解释。

**参数**：
- `x`: 观察到的事件数
- `n`: 观察单位数
- `lambda0`: 总体均数
- `alpha`: 显著性水平

**返回值**：
- `float`: 检验效能值

### 结果计算函数 - cal_result_pois3

生成泊松分布样本均数与总体均数比较统计分析的完整报告字典。此函数整合了泊松分布样本均数与总体均数比较的所有关键指标，生成标准化的字典格式报告，适用于临床研究报告的需求，提供全面的比较分析结果。

**参数**：
- `param`: BaseParamPois3对象，包含total_posi_p, sample_size, sample_posi_num

**返回值**：
- 包含泊松分布样本均数与总体均数比较统计分析指标的字典，包括：
  - `input_parameters`: 输入参数信息
  - `exact_test`: 精确检验结果
  - `normal_approximation_test`: 正态近似检验结果
  - `power`: 检验效能
  - `validation`: 有效性验证
  - `interpretation`: 结果的专业解释

## 结果解读注意事项

1. **P值解释**：在零假设成立的前提下，观察到当前或更极端结果的概率
2. **显著性水平**：通常使用α=0.05作为判断标准
3. **单双侧选择**：根据研究目的选择合适的检验方向
4. **临床意义**：统计显著性不等同于临床重要性
5. **检验效能**：注意小样本可能导致的低效能问题

## 使用示例

```python
from app.stats.base.base_stats_pois3 import cal_result_pois3
from app.schemas.request_data.base_param import BaseParamPois3

# 创建参数对象
param = BaseParamPois3(
    total_posi_p=2.0,      # 总体均数（率）为2.0
    sample_size=100,       # 观察单位为100
    sample_posi_num=250    # 观察到的事件数为250
)

# 计算泊松分布样本均数与总体均数比较
result = cal_result_pois3(param)

# 查看结果
print(f"输入参数: {result['input_parameters']}")
print(f"精确检验结果: {result['exact_test']}")
print(f"正态近似检验结果: {result['normal_approximation_test']}")
print(f"检验效能: {result['power']}")
print(f"有效性验证: {result['validation']}")
print(f"结果解释: {result['interpretation']}")
```
